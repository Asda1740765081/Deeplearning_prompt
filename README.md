# DeepPrompt Lab

面向深度学习研究者的中文 Prompt 提示词网站，覆盖论文阅读、论文写作、实验设计、代码研发、研究创新和组会汇报等任务。

## 功能

- 17 个结构化研究 Prompt 模板
- 分类浏览与关键词搜索
- 填写研究信息后生成完整提示词
- 一键复制、收藏和深色模式
- 桌面端与移动端响应式布局
- 纯静态网站，无需后端或数据库

## 使用

直接打开 `index.html`，或通过 GitHub Pages 部署为公开网站。

收藏和主题设置仅保存在访问者自己的浏览器中。

## GitHub Pages

在仓库的 `Settings > Pages` 中，将发布来源设置为 `Deploy from a branch`，分支选择 `main`，目录选择 `/ (root)`。
